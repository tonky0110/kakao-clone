flex 자동으로 그려주는 그리드가 없었음. 부모element에게 속성을 부여. default 속성: inline-block flex-direction: row, column(자식wlement의 방향 지정.), row-reverese, column-reverese justify-content: flex-direction과 같은 방향으로 정렬 설정. 1) center, 2) space-between, 3) space-around align-items: flex-direction 수직방향의 정렬 설정 1) flex-start, 2) flex-end, flex-direction: row --> justify-content(수평) && align-item(수직) flex-direction: column --> justify-content(수직) && align-item(수평) flex-wrap: 1) wrap: element의 사이즈를 고정하고, 개행처리. 2) no-wrap: element의 사이즈를 조정해서 '한'줄로 표현 3) wrap-reverse: row/column의 순서가 반대
